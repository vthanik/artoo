# artoo: Lossless CDISC-Native Input and Output for Clinical Datasets

Reads and writes clinical-trial datasets losslessly across SAS XPORT
(XPT), CDISC Dataset-JSON, and Apache Parquet, applying a specification
to produce submission-ready SDTM and ADaM datasets. A single canonical
metadata model carries labels, CDISC data types, lengths, SAS display
formats, controlled-terminology references, and sort keys identically
across every format, so conversion between any two formats is lossless
by construction. Pure R and lightweight, with no external 'SAS' or
'Java' runtime.
